Welcome to Hexoopen in new window! This is your very first post. Check documentationopen in new window for more info. If you get any problems when using Hexo, you can find the answer in troubleshootingopen in new window or you can ask me on GitHubopen in new window.

# 前期准备

要使用hexo，首先你需要检测电脑上是否已经安装下列应用程序：

Node.jsopen in new windowGitopen in new window

如果已经安装上述必备程序，那么接下来只需要使用 npm 即可完成 Hexo 的安装。

npm install -g hexo-cli

# Quick Start

# Initial project

hexo init blog

# Create a new post

hexo new "My New Post"

More info: Writingopen in new window

# Run server

hexo server

Or

hexo s

More info: Serveropen in new window

# Deploy

# Install git deploy plugin

npm install hexo-deployer-git --save

# Configure

open file "_config.yml" in the project at first level, and configure deploy node.

deploy:
type: git
repo: git url
branch: master

# Clean browser cache

hexo clean

# Generate static files

hexo generate

Or

hexo g

More info: Generatingopen in new window

# Deploy to remote sites

hexo deploy

Or

hexo d

More info: Deploymentopen in new window

There is no details about github page configure, about more info please google.
